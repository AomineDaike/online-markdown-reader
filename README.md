# Online markdown reader
Markdown 文件在线阅读器， 使用 [marked](https://github.com/chjj/marked) 解析文件。

## 功能
- [x] 文件拖拽
- [x] `highlight.js` 代码高亮
- [x] `MathJax` 公式
- [x] Task list - `- [ ]` `- [x]`
- [ ] 图片上传显示
- [ ] html 代码下载
- [ ] 自动加载 Markdown 文件
